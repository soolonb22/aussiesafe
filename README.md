# AussieSafe 🛡️

**Privacy-first Australian kids online safety browser extension**

Made in Mackay, Queensland — designed to protect kids while building real resilience, not just blocking.

### Why AussieSafe?

Most parental control tools feel like surveillance.  
AussieSafe is different.

- **100% local-first** — Everything stays on your device. No chat scanning, no data sent to the cloud.
- **Privacy by design** — Compliant with Australian Privacy Principles and eSafety guidelines.
- **Teaches resilience** — Gentle Safety Co-Pilot nudges + transparent practice scenarios instead of hidden monitoring.
- **Made for Australian families** — Especially regional QLD families who want trustworthy, no-nonsense tools.

### Key Features

#### Free Tier (Core Protection)
- Smart website and category blocking
- Time-based restrictions (bedtime & school hours)
- Local keyword flagging for grooming red flags (parent editable)
- Age profile setup
- Family Safety Pact (shared family rules)
- Direct eSafety Commissioner help links

#### Premium Features (Coming Soon)
- Safety Co-Pilot — gentle real-time safety nudges on risky pages
- Resilience Score — transparent practice scenarios with parent insights
- Deepfake Quick Check — on-demand scan for suspicious images/videos
- Community Aussie Safe List — parent-voted blocklist for local threats
- Kid Mode Override + safety rewards system

### Tech Stack
- Chrome & Firefox Manifest V3
- Pure JavaScript (no heavy frameworks)
- All data stored locally using `chrome.storage`
- Lightweight content scripts for visible page analysis only

### Installation

1. Download or clone this repository
2. Open Chrome → go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **"Load unpacked"** and select the `aussiesafe-extension` folder
5. Pin the extension and open Settings to customise

Firefox users can load it temporarily via `about:debugging`.

### Privacy Commitment
- No data collection
- No cloud syncing (unless explicitly chosen in premium)
- No AI monitoring of private conversations
- Everything runs locally on your device
